# oyun-proje

Vanilla HTML ve JavaScript ile Tetris oyunu.

## Calistirma

1) Depodaki `index.html` dosyasini tarayici ile acin (cift tik veya yerel sunucu).  
2) Kontroller: ok tuslari hareket/rotasyon, Space sert dusus, C parca tut/degistir, P duraklat, R sifirla.  
   UI butonlariyla duraklat veya yeniden baslat da yapabilirsiniz.

Ek kutuphane gerekmez.
